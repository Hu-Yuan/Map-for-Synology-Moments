# 地图模式-群晖Moments

[中文](https://github.com/Hu-Yuan/Map-for-Synology-Moments/blob/master/README-cn.md) | [English](https://github.com/Hu-Yuan/Map-for-Synology-Moments/blob/master/README.md)

一个用于群晖NAS的Web应用，可以让你通过地图模式来查看Moments中的照片。

## 特性

- 通过地图快速找到某个地方拍摄的照片
- 一目了然照片是在什么地方拍摄的
- 流畅的使用体验
